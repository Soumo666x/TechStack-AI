## 🧠 Overview

TechStack AI is a web-based tool that helps startups choose the most suitable technology stack based on their **stage, team size, budget, and sector**.

Instead of generic advice, it provides intelligent, practical, and India-focused recommendations to help founders build faster and smarter.

---

## ✨ Features

- 🎯 Tailored tech stack recommendations  
- 💰 Monthly cost breakdown (INR-based)  
- 🇮🇳 India-specific integrations (Razorpay, Aadhaar, ABDM, etc.)  
- ⚠️ Real-world constraints & trade-offs  
- 📈 Scaling roadmap from MVP → Scale  
- 📊 Dynamic recommendation engine  
- 🧠 Smart rule-based decision logic  
- 🎨 Clean and modern UI  
- ⚡ Instant results with confidence score  
- 📋 Copy full report functionality

## ⚙️ How It Works

1. Select:
   - Startup Stage  
   - Team Size  
   - Budget  
   - Sector  

2. Click **Generate Recommendation**

3. Get:
   - Recommended tech stack  
   - Cost estimation  
   - Scaling roadmap  
   - Real-world insights
